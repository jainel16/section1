CACHE MEMORY

Step 0: ____________________________________________

Step 1: Data requested will first enter the CPU.

Step 2: The CPU checks if it already has the requested data.

Step 3: Cache informs the CPU the data is not stored.

Step 4: CPU sends request to RAM to get the data.

Step 5: If RAM does not have it, data is fetched from HDD.

Step 6: HDD sends data to CPU in fragments via RAM.

Step 7: Cache reads remaining fragments.

Step 8: Cache sends fragments to CPU.

Step 9: CPU completes the process and delivers requested data.
